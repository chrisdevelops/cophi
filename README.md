# Cophi
#### PHP Compiler... Written in PHP.

##### Roadmap

1. Complete chroot jailer component
2. Create extension loader
3. Optimise AST & Opcache
4. Create Cophi CLI
5. Package to PHAR

##### Requirements
- pkg-config
- libjpeg-dev
- libpng-dev
- libcurl4-openssl-dev
- libxml2-dev
- libtidy-dev
- php*.*-dev
- jailkit (https://olivier.sessink.nl/jailkit/)
